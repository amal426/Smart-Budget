#المساعد المالي الذكي (Smart Budget)

&gt; **مساعد مالي ذكي مخصص للفئات التي تجد التطبيقات المصرفية التقليدية معقدة** — يقدم توصيات مالية شخصية بلغة بسيطة ومحاكي حياة مالية تفاعلي.

---

# المشكلة التي نحلها

العديد من الفئات (كبار السن، ذوي الاحتياجات الخاصة، المبتدئون في التعامل الرقمي) يواجهون صعوبة في:
- فهم المصطلحات المصرفية المعقدة
- اتخاذ قرارات مالية سليمة
- ربط قراراتهم اليومية بمستقبلهم المالي

---

## المميزات الرئيسية

### 1.توصيات مالية شخصية بلغة بسيطة
- **أمثلة عملية**: "لو وفرت 200 ريال شهرياً، خلال 5 سنوات تقدر تفتح مشروعك الصغير"
- ترجمة المصطلحات المالية لغة يومية مفهومة
- نصائح مخصصة بناءً على وضع المستخدم المالي

### 2.محاكي الحياة المالية (Financial Life Simulator)
- ربط القرارات المالية الحالية بتأثيرها المستقبلي
- سيناريوهات تفاعلية: "ماذا لو وفرت؟" vs "ماذا لو أنفقت؟"
- تصور بصري للتأثير المركب للتوفير على المدى الطويل

### 3.واجهة سهلة الاستخدام
- تصميم بسيط وبديهي
- دعم اللغة العربية بالكامل
- تجربة مستخدم محسّنة للفئات المستهدفة

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
